Welcome to Classifier API

About This Project

This is a simple API that performs machine learning classification. It processes input data and provides classification results based on pre-loaded learning data. The API is built with Flask and is deployed on Heroku.

Endpoints

  • [GET] / - Homepage (this page)
  • [GET] /server_ready - Check if the server is ready
  • [POST] /classify - Classify the input text
  • [POST] /generate_inputs - Generate inputs based on classification type

Usage of the API

Keep in mind this API is built with freely available datasets, so it might occasionally misclassify things.

It's also running with just one worker to keep server memory low, but feel free to try it out!
