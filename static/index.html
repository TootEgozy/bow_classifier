Welcome to My API!

About This Project

This is a simple API that performs machine learning classification. It processes input data and provides classification results based on pre-loaded learning data. The API is built with Flask and is deployed on Heroku.

API Endpoints

  • /classify - Classify the input text
  • /generate_inputs - Generate inputs based on classification type
  • /server_ready - Check if the server is ready

Powered by Flask and Heroku. API Version 1.0
